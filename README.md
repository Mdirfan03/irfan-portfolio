
# Irfan Portfolio (React + Vite)

This is a production-ready React portfolio starter built with Vite.
It includes:
- Home (hero + profile)
- About (skills & tools)
- Projects (case studies linking to Behance)
- Contact
- Resume included in /public/Irfan-Resume.pdf

## Run locally

Install dependencies:
```
npm install
```

Run dev server:
```
npm run dev
```

Build:
```
npm run build
```
